## document.removeAllFilters()

#### Availability

Flash 8.

#### Usage

document.removeAllFilters()

#### Parameters

None.

#### Returns

Nothing.

#### Description

Method; removes all filters from the selected objects.

#### Example

```javascript
The following example removes all filters from the selected objects:
fl.getDocumentDOM().removeAllFilters();

```
